# Merertu Philipos Portfolio

Design and Development of an Interactive Full-Stack Portfolio Website Using React and Node.js.

## Project Overview
This project aims to design and develop an interactive, modern, and responsive portfolio website reflecting a passion for full-stack development and cybersecurity.

## Tech Stack
- **Frontend**: React
- **Backend**: Node.js
- **Styling**: Vanilla CSS
- **Animations**: Interactivity and motion libraries

## Project Structure
- `frontend/`: React components and UI
- `backend/`: Node.js Express server for form handling and future expansion
